# Course 2 - Module 1 - Overview of Cloud Computing

## Key Concepts
- Cloud computing = on-demand access to shared computing resources over the internet
- 5 essential characteristics: on-demand self-service, broad network access, resource pooling, rapid elasticity, measured service
- 3 service models: IaaS (infrastructure), PaaS (platform), SaaS (software)
- 3 deployment models: Public, Private, Hybrid
- Cloud accelerates emerging tech: IoT, AI, Blockchain
- Business case: cost efficiency, scalability, flexibility — no need to own physical hardware

## New Terms
- IaaS: you manage apps/data, cloud provides servers/storage/networking (e.g. AWS EC2)
- PaaS: you manage apps/data, cloud provides everything else including OS (e.g. Heroku)
- SaaS: cloud manages everything, you just use the software (e.g. Gmail, Spotify)
- Hybrid Cloud: mix of public and private clouds working together

## Summary
Cloud computing lets businesses access computing resources on demand without owning
physical hardware. The three service models (IaaS, PaaS, SaaS) differ in how much
the cloud manages vs how much you manage. Public, private, and hybrid deployments
offer different levels of control and cost.
