# Course 1 - Module 2 - Introduction to software development

## Key Concepts
- Websites work through front-end and back-end technologies communicating together
- Front-end handles users reactivity and responsive UI
- Back-end handles the business logic, security and database access
- Version control enables collaboration and track code changes

## New Terms
- Software stack: combination of technologies used together to build an app
- CI/CD pipeline: automates building, testing and deployment of code continuously 
- Framework: pre-built structure that speeds up development
