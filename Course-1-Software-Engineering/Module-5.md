# Course 1 - Module 5 - Job oppurtunities and skillset

## Key Concepts
- Software engineers design, develop, maintain and update software systems
- On-the-job learning is constant — tech changes fast, skills must keep evolving
- Success needs both hard skills (coding) and soft skills (communication, teamwork)
- High demand field with flexible career paths — technical or managerial track
- Many job titles exist under "software engineer" — each with different focus areas
- Code of Ethics has 8 principles: public, client/employer, product, judgment, management, profession, colleagues, self

## New Terms
- Software Engineering Code of Ethics: 8 guiding principles for professional conduct in the field

## Summary
Software engineers need more than just coding skills — soft skills and continuous
learning matter just as much. The field offers both technical and managerial career
paths. The Code of Ethics sets professional standards covering responsibility to the
public, employer, product quality, and colleagues.
